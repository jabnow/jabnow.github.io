    About·What is this
    About

    A small filing cabinet to keep track of past projects and WIPs.

    The home view is a force-directed graph of every project, grouped around a center node. Click a project to edit it; drag any node to rearrange the layout.

    The list view supports search, category folders, and sort. Everything you add or change is stored in this browser's localStorage, so it sticks around between visits on this device.

    Stack

    D3 Force Graph
    Browser localStorage

    Tips

    Clear all data
    Open DevTools → Application → Local Storage → delete the jabnow_projects_v1 key. The next load will reseed with defaults.
    DevTools
    Per-device storage
    Data lives on this browser only. To sync across devices you'd need to add a backend later.
